Endianness is the order in which bytes are stored in memory.
## Little Endian
In a little-endian system, the least significant byte (the "smallest" part of the number) is stored first, at the lowest memory address. ==This is the format used by most modern computers.==
- Let's say you have the hexadecimal number `0x12345678`, the least significant byte (`0x78`) is stored first (at the lowest memory address).

## Big Endian
In a big-endian system, the most significant byte is stored first, at the lowest memory address.
- Let's say you have the hexadecimal number `0x12345678`, the most significant byte (`0x12`) is stored at the lowest memory address.
